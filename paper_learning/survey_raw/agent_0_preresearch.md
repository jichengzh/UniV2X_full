# Phase 0 预调研结果

## 发现的协同方向 (12个)

1. LLM/大模型推理加速 — 高
2. 量化与低精度计算协同 — 高
3. 存内计算(CIM/PIM)协同优化 — 中
4. 稀疏性利用与加速 — 中
5. HW-NAS — 中
6. DNN编译器与代码生成 — 高
7. Chiplet/异构多芯片架构 — 低→中
8. FPGA上的DNN加速 — 中
9. 端侧/边缘AI加速 — 中
10. 全栈端到端协同框架 — 低→中
11. 训练加速与分布式协同 — 中
12. 安全/可靠性协同 — 低

## PM决策: 最终分类 (5类 + 1固定类)

| Agent | 大类 | 覆盖子方向 |
|-------|------|-----------|
| Agent-1 | LLM/Transformer专用加速 | 1, 4(稀疏注意力部分) |
| Agent-2 | 模型压缩与高效表示的硬件协同 | 2, 4(剪枝), 5 |
| Agent-3 | 编译与系统软件栈优化 | 6, 10, 11(部分) |
| Agent-4 | 新型计算架构 | 3, 7, 8 |
| Agent-5 | 自动驾驶/V2X部署加速 | 固定章节 |
